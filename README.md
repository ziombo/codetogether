# CodeTogether
CodeTogether is a website that allows people to pair programme in real time.

# Built With:
1. C#
  * ASP.NET WebPages
  * SignalR
2. HTML
3. CSS
4. JavaScript
  * jQuery
5. [Ace Editor](https://ace.c9.io/)
